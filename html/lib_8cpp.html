◆ Count_Devices()

int Count_Devices ( FILE * f)

Retorna la totalidad de dispositivos conctados en la red.

Esta función toma como parametro el manipulador del archivo, con el primer header de uint16_t, obtengo el Lower Level Device Count, salto los otros dos uint16_t del header que no me interesan por el momento, salto los Lower Level Device ID con un fseek() y cuento con la variable countHeaders la cantiadad total de dispositivos.

Parameters
fManipulador del archivo archivo network_structure.dat.
Returns
countHeader, El número total de dispositivos conectados en la red.
